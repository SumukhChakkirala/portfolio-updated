Face Recognition Attendance System

this is a technology-powered attendance system designed to solve a real problem. the usual attendance system in schools and colleges consumes a lot of time that could be spent on more productive activities. the idea is simple: a webcam captures all the students present and lets the system mark attendance automatically, no manual intervention needed.

this project uses python, opencv, the face_recognition library, and twilio for messaging. the system captures live video from a webcam and detects faces in real time. opencv draws bounding boxes around detected faces while the face_recognition library matches them against pre-encoded facial features stored in a pickle file. this encoding approach makes the matching process fast and efficient, even when processing multiple faces simultaneously. the backend runs on django and connects to a mysql database that stores student names, registration numbers, attendance logs, and timestamps. this database structure makes it easy to pull up daily reports, track attendance patterns over time, and generate analytics for administrators or faculty.

one key feature is the automated notification system using the twilio api. whenever a student is marked absent, the system automatically sends a whatsapp message to designated contacts. this keeps students informed about their attendance status in real time, helping them plan how many classes they can afford to miss while still meeting the minimum attendance requirement. it's practical and solves the common problem of students not knowing their attendance until it's too late. the whole system comes together as a complete automation solution that saves time for teachers, keeps students informed, and maintains accurate records without the hassle of traditional attendance methods.
